# Expense Tracker (Tkinter)

A simple desktop expense tracker built with Python and Tkinter that helps users record, categorize, and monitor daily expenses.

## Features

- Add and manage expenses  
- Categorize spending (Food, Travel, Bills, etc.)  
- View expense history  
- Simple and user-friendly GUI  
- Lightweight desktop application  

## Built With

- Python  
- Tkinter
